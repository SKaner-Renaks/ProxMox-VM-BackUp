# Changelog

## v 0.02.00
- **Изменение логики `vm_config`**:
  - `0`: Пропустить VM (не копировать, не ротировать).
  - `-1`: Удалить все локальные копии этой VM.
  - `> 0`: Хранить указанное количество последних копий.
- **Предварительная информационная сводка**:
  - Отображение общего количества VM в конфигурации.
  - Отображение количества найденных бэкапов на сервере.
  - Расчет суммарного объема планируемых данных.
  - Список пропущенных VM (со значением 0).
- **Проверка свободного места**:
  - Расчет количества возможных полных копий.
  - Три уровня предупреждений: Достаточно места (зеленый), Мало места (оранжевый), Недостаточно места (красный + выход).
  - Настраиваемый параметр `MIN_FREE_COPIES_WARNING`.
- **Итоговая статистика**:
  - Добавлен расчет и вывод общего объема реально скопированных данных.
- **Расширение цветовой схемы**:
  - Добавлены `ORANGE`, `BRIGHT_GREEN` и `BOLD` (жирный шрифт).
- **Улучшения UI**:
  - Использование эмодзи (✓, ⚠, ✗) для визуального выделения статусов.
